# 🎤 Emotion Recognition from Speech

A Machine Learning project that detects human emotions from speech audio.

## 🚀 Features
- MFCC feature extraction
- Random Forest model
- Streamlit web app
- Real-time audio prediction

## 🧠 Emotions
Happy, Sad, Angry, Fearful, Neutral, Disgust, Surprised

## ⚙️ Tech Stack
- Python
- Librosa
- Scikit-learn
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python train.py
streamlit run app.py
